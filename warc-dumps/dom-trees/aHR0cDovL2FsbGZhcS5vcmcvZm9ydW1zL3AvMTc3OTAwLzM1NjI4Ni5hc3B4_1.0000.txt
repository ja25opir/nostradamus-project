<div class="richText">
The other way you can do this is to have a command in an logon script that writes out to a text file on a network share. For example<br><br>echo Logon %username%, %computername%, %date% , %time%. &gt;&gt;&nbsp;\\servershare\foldershare\<wbr>login.csv<br><br>That way when they login, it automatically sets up an entry into the login.txt file. <br>Saves reading event logs.<br>You can also extend this with a log off script, which could be setup to do the same.<br><br>I have acutally gone even further and setup a logon and logoff script that updates a database table, which can then be viewed in a asp page, listing the user's logon habbits etc.<br><br>The cool thing about this is the information is always there and won't get overwritten.<br><br>Regards<br><br><br></div> <!-- richText -->
