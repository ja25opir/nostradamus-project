<h2 id="post-title"><a href="/blog/archive/2003/11/14/shadows/" title="A permalink for this post">Easy CSS drop shadows</a></h2>

<p>A few people have noticed that the drop shadows I use[d] on images here aren’t part of the pictures themselves, but are added on through the web equivalent of Magic Fairy Dust — CSS. I thought I’d explain how it’s done in case anyone else wanted to know.</p>

<p>I have two different implementations of this technique: one is for images that are used for decorating a post; one is for images that are used to illustrate a post. Both can be seen in the <a href="http://demo.1976design.com/dropshadows/" title="View the demo">demo</a>.</p>

<p>Now then, before we look at the code, have a nose at this picture; it should give you an overview of what actually happens. (I’ve used a blue background so you can see what’s what, and only swapped to a white background at the end for the full effect.)</p>

<div class="image" style="width: 309px;"><img src="/blog/images/49a.gif" width="297" height="673" alt="Graphical overview of this technique"></div>



<h3>The mark-up</h3>

<p>This is the code to insert into your web pages.</p>


<h4>For decorative images</h4>

<ol class="code">
<li><code>&lt;div class="img-dec"&gt;&lt;img src="cat.gif" width="260" height="200" alt="A stupid waving cat" /&gt;&lt;/div&gt;</code></li>
<li class="source">Download this code: <a href="/blog/code/49a.txt" title="Download the above code as a text file">/code/49a.txt</a></li>
</ol>



<h4>For illustrative images</h4>

<ol class="code">
<li><code>&lt;div class="img-ilus" style="width: 270px;"&gt;&lt;img src="cat.gif" width="260" height="200" alt="A stupid waving cat" /&gt;&lt;/div&gt;</code></li>
<li class="source">Download this code: <a href="/blog/code/49b.txt" title="Download the above code as a text file">/code/49b.txt</a></li>
</ol>



<h3>The drop shadow image</h3>

<p>Rather than spend ages trying to describe how I made the drop shadow, I’ll just provide <a href="http://download.1976design.com/dropshadows/" title="Download the demo files">the Photoshop file and GIF files for you to download</a>, and you can see for yourself.</p>

<p>You’ll notice that the image is actually pretty big, at 800×800 pixels. I originally started off with a much smaller file, planning never to display images above 300×200 pixels on my blog, but I soon found that I out grew this limit, and the drop shadow gif just wasn’t large enough. So, knowing that such a simple image is actually pretty small in terms of file size (the 800×800 version I use now is only 3.47KB) I thought I’d save myself future hassle and produce a larger-than-I'll-ever-need version.</p>


<h4>Drop shadows for different coloured sites</h4>

<p>My drop shadow is made using a white background, because that’s the colour of my site. If you find that you need yours to sit on top of lots of different colours (maybe you have a style-switcher on your site), then you could always set a transparent background and save the image as a PNG instead. (Be sure to check the current browser support for this though.)</p>


<h3>The CSS</h3>

<p>Pop these rules into your style sheet.</p>


<h4>For decorative images</h4>

<ol class="code">
<li><code>div.img-dec {</code></li>
<li class="tab1"><code>background: url('shadow.gif') no-repeat bottom right;</code></li>
<li class="tab1"><code>clear: right;</code></li>
<li class="tab1"><code>float: right;</code></li>
<li class="tab1"><code>margin: 0 0 15px 25px;</code></li>
<li class="tab1"><code>padding: 0;</code></li>
<li class="tab1"><code>position: relative;</code></li>
<li><code>}</code></li>
<li>&nbsp;</li>
<li><code>div.img-dec img {</code></li>
<li class="tab1"><code>background-color: #fff;</code></li>
<li class="tab1"><code>border: 1px solid #a9a9a9;</code></li>
<li class="tab1"><code>display: block;</code></li>
<li class="tab1"><code>margin: -5px 5px 5px -5px;</code></li>
<li class="tab1"><code>padding: 4px;</code></li>
<li class="tab1"><code>position: relative;</code></li>
<li><code>}</code></li>
<li class="source">Download this code: <a href="/blog/code/49c.txt" title="Download the above code as a text file">/code/49c.txt</a></li>
</ol>



<h4>For illustrative images</h4>

<ol class="code">
<li><code>div.img-ilus {</code></li>
<li class="tab1"><code>background: url('shadow.gif') no-repeat bottom right;</code></li>
<li class="tab1"><code>margin: 10px 5px;</code></li>
<li class="tab1"><code>padding: 0;</code></li>
<li class="tab1"><code>position: relative;</code></li>
<li><code>}</code></li>
<li>&nbsp;</li>
<li><code>div.img-ilus img {</code></li>
<li class="tab1"><code>background-color: #fff;</code></li>
<li class="tab1"><code>border: 1px solid #a9a9a9;</code></li>
<li class="tab1"><code>margin: -5px 5px 5px -5px;</code></li>
<li class="tab1"><code>padding: 4px;</code></li>
<li class="tab1"><code>position: relative;</code></li>
<li class="tab1"><code>vertical-align: bottom;</code></li>
<li><code>}</code></li>
<li class="source">Download this code: <a href="/blog/code/49d.txt" title="Download the above code as a text file">/code/49d.txt</a></li>
</ol>


<p>And that’s it. It should be pretty easy to get that up and running on your own site, just make sure the CSS paths to the shadow image are correct. Good luck!</p>

<p class="note"><ins datetime="2003-11-15T15:37:00-00:00">Update: Thanks to <a href="http://www.sujag.de/" title="Go to www.sujag.de">Susanne Jaeger</a> for helping me get this working in all three major browsers after <a href="http://www.xebit.net" title="Go to www.xebit.net">Tony</a> pointed out a bug.</ins></p>


<h3>Alternative methods</h3>

<p>2003/11/14: I see that both <a href="http://www.designdetector.com/tips/shadow.html" title="Go to www.designdetector.com">Chris Hester</a> and <a href="http://blog.tom.me.uk/" title="Go to blog.tom.me.uk">Tom Gilder</a> have come up with their own solutions (very nice too), but while theirs rely on adding in six empty <code>&lt;div&gt;</code>s, this technique doesn’t require any additional markup, and that’s what I was aiming for.</p>

<p>2004/03/30: <a href="http://overcaffeinated.net/" title="Go see Sergio’s homepage">Sergio Villarreal</a> has <a href="http://www.alistapart.com/articles/cssdropshadows/" title="Go see Sergio’s effort">extended this technique</a> (albeit by using extra markup) in an article on <a href="http://www.alistapart.com/" title="Go to www.alistapart.com">A List Apart</a>. <a href="http://wubbleyew.com/" title="Go see Phil’s homepage">Phil Baines</a> then took Sergio’s code and <a href="http://wubbleyew.com/tests/dropshadows.htm" title="Go see Phil’s effort">simplified it</a>. It's worth taking a look at both of their explanations.</p>
