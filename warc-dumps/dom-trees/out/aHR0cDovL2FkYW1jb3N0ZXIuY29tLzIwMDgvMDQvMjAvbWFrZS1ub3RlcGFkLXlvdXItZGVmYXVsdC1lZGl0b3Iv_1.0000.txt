			<h1 class="post-title" id="post-93">Make Notepad++ your default&nbsp;editor</h1>
			<p class="post-metadata">2008.04.20 in <a href="http://en.wordpress.com/tag/computerssoftware/" title="View all posts in computers/software" rel="category tag">computers/software</a> | Tags: <a href="http://en.wordpress.com/tag/default/" rel="tag">default</a>, <a href="http://en.wordpress.com/tag/editor/" rel="tag">editor</a>, <a href="http://en.wordpress.com/tag/executable/" rel="tag">executable</a>, <a href="http://en.wordpress.com/tag/microsoft/" rel="tag">microsoft</a>, <a href="http://en.wordpress.com/tag/notepad/" rel="tag">notepad</a>, <a href="http://en.wordpress.com/tag/replacement/" rel="tag">replacement</a>, <a href="http://en.wordpress.com/tag/text/" rel="tag">text</a>, <a href="http://en.wordpress.com/tag/text-editor/" rel="tag">text editor</a>, <a href="http://en.wordpress.com/tag/windows/" rel="tag">windows</a>, <a href="http://en.wordpress.com/tag/word-processor/" rel="tag">word processor</a>			</p>
		</div>
		<div class="post-content">
			<script type="text/javascript" src="http://s0.wp.com/wp-content/plugins/adverts/adsense.js?m=1253160243g&amp;1"></script><p>If you ever use Notepad, Windows’ native text editor, you should upgrade to <a href="http://notepad-plus.sourceforge.net/uk/site.htm">Notepad++</a>. Immediately. Seriously, every moment that you don’t is one more moment of sheer misery you don’t even realize you’re going through! Even if you don’t normally use a text editor, you might as well get a nice one. Just in case.</p>
<p>So what’s the difference between a text editor and a word processor (like Microsoft Word)? Well, the name pretty much says it all. A text editor does nothing but manipulate text, just by adding or removing characters. A text editor can also be used to read any file-type, though if you open something like a jpeg you’ll have what looks like complete gibberish. Editors do absolutely no formatting. No font changes, size changes, italics, bold, etc etc. A word processor adds all of this fancy formatting, and usually uses all kinds of added code for both proprietary and useful reasons. For a good example, open up a Word document with Notepad. You’ll see the text of your document, but there is all kinds of crap between words, letters, paragraphs, and at the header and footer.</p>
<p>You can actually edit any kind of file you want with a text editor, but chances are you’ll just make the file stop working properly. It can be fun, however, to explore different files and see what kind of stuff is added by the software that runs them. You may even find that some files store and hide some information about you…</p>
<p><span id="more-93"></span></p>
<p>If you’ve made it this far, the next question is “what’s the difference between Notepad and Notepad++?”</p>
<p>First off, Notepad++ is completely <a href="http://en.wikipedia.org/wiki/Open_source">open source</a>, free, and community-maintained. There are all kinds of plugins that come with it, and others you can download to add functionality. At base, ++ does a lot of things that plain old Notepad doesn’t do. It does spell checking, for one. It also does syntax highlighting which means that ++ will read the extension of the file (the part that says <strong>.txt</strong>, <strong>.doc</strong>, <strong>.jpg</strong>, etc) and then add colors and underlining in some sort of sensible fashion. This is extremely useful for any kind of coding such as LaTeX, HTML, and other computer languages.</p>
<p>Perhaps even more useful (for most people), ++ has a tabbed interface, and can be set up to remember what tabs it previously had open (and is set this way by default). So, you can have 10 different documents open at once, all in the same window, and if you somehow accidentally close it you can just reopen the program and everything will be there, safe and sound. Like magic.</p>
<p>If you’ve been sold, here are the directions to make Notepad++ your default text editor (Read my <a href="http://adamcoster.com/disclaimer/">Disclaimer</a> first!):</p>
<p>Download the Windows <a href="http://downloads.sourceforge.net/notepad-plus/npp.4.8.5.Installer.exe?modtime=1207960310&amp;big_mirror=0">executable</a> (direct link via sourceforge) and then run through the installation. That’s all that is required if you want Notepad++, but if you want to make it your default editor there are a few more things to do:</p>
<p>Download <a href="http://notepad-plus.sourceforge.net/commun/misc/NppLauncher.bin.zip">this launcher</a> (direct link via Notepad++ <a href="http://notepad-plus.sourceforge.net/uk/site.htm">download page</a>). It comes as a .zip with a text (.txt) file and an executable (.exe) file inside. Unzip the zip. The text file gives you the directions for making ++ your default, so have at it if you don’t want my extra details. For everyone else, fire up Explorer. This can be done by double-clicking any folder around or by going to the <strong>Start menu</strong>, clicking <strong>Run</strong>, and then typing “explorer” into the box and hitting Enter. Now copy this text</p>
<p style="text-align:center;"><em>c:\windows\system32\dllcache</em></p>
<p style="text-align:left;">and paste it into the address bar at the top of the Explorer window. Hit Enter, and you’ll be inside the folder (also referred to as a “directory”). You’ll see all kinds of stuff in blue text– make sure that you <strong>do not delete</strong> or rename <em>any</em> of these (that might very well screw up your computer). Just in case you change your mind in the future and want Notepad back, start typing “n o t …” until a file called notepad.exe is highlighted. Right click it and then click “Rename.” Change the name to <strong>notepad.exe.bak</strong>, or anything you can easily remember. This just makes it so that Windows can’t find it, but you can just change the name back to what it was to make it findable again. By the way, this might make Windows upset so that it will warn you or recommend that you quit fooling with its guts. Don’t listen (choose whichever option means “I’ll do what I want!”).</p>
<p style="text-align:left;">Now go back to your Desktop, or wherever you left the ++ executable, and copy it. Paste it into the <em>c:\windows\system32\dllcache</em> folder that you just opened. Notice that you have simply replaced the original Notepad.exe with something that has the same name, but contains different stuff. So Windows is going to look for Notepad.exe, and then launch the new file you stuck in there instead of the old one (which now has a different name). How sneaky of us!</p>
<p style="text-align:left;">Unfortunately, you have to do this process two more times (Windows wants to make sure you are absolutely positive you want to do this). So, go to <em>c:\windows\system32</em> in Explorer and rename Notepad.exe, then paste in the new version. Do it again in<em> c:\windows</em>. Finally! Windows will throw some more warnings at you, and ask if you want to revert to the original files, threaten to murder your first-born child, etc. Again, tell it you’ll do what you want.</p>
<p style="text-align:left;">Now you have replaced Notepad with Notepad++. By default, every time you double-click a .txt file you’ll end up inside Notepad++. Easy, eh? There will also be an “Edit with Notepad++” option added to your right-click context menu. Enjoy your new and improved text editor!</p>
